# Quiet Sleep Landing

Static landing page for Quiet Sleep.

## Deploy

This site has no build step. Deploy the repository root to:

- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel

For GitHub Pages, publish from the `main` branch and root folder.
